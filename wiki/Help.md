glassic commands
----------------

    npm install

      - updates glassic
      - installs dependencies
      - compiles according to **config.json**
      - prints help

    npm run glassic

      - alias for npm install

    npm run demo:linux

      - installs linux dependencies
      - runs the application

    npm run demo:android

      - installs android dependencies
      - runs the clean task
      - compiles the application
      - prints android help
